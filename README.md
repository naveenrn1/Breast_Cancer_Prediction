# Breast Cancer Prediction System

AI-powered Breast Cancer Prediction System using:

- Machine Learning
- Flask
- SQLAlchemy
- PyTorch
- Image Detection
- PDF Report Generation

## Features

- Tabular Data Prediction
- Image-based Cancer Detection
- Prediction History
- Dashboard Analytics
- Downloadable PDF Reports

## Technologies Used

- Python
- Flask
- Scikit-learn
- PyTorch
- SQLite
- Bootstrap

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python app.py
```
